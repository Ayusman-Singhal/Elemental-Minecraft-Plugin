# Elemental ServerCore

## Overview
ServerCore is a Minecraft server plugin designed to provide core functionality and utilities for managing and enhancing server operations.

## Features
- Core server management tools
- Plugin utilities and helpers
- Configuration management
- Performance monitoring
- Command handling

## Installation
1. Download the latest release
2. Place the `.jar` file in your server's `plugins/` directory
3. Restart your Minecraft server
4. Configuration files will be generated in `plugins/ServerCore/`

## Configuration
Edit the configuration files in `plugins/ServerCore/config.yml` to customize plugin behavior.

## Commands
- `/servercore help` - Display available commands
- `/servercore reload` - Reload plugin configuration
- `/servercore status` - View server status

## Requirements
- Java 11 or higher
- Spigot/Paper 1.16+
